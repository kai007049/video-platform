## 💡 常见问题

**Q: 样式还是没出现？**
A:

1. 确保你按了 Ctrl+Shift+R 硬刷新
2. 检查浏览器控制台是否有错误（F12）
3. 检查开发者工具的 Elements 标签，看 inline style 是否在 HTML 上

**Q: 动画不流畅？**
A:

1. 检查浏览器性能（打开 Performance 标签）
2. 减少同时打开的标签页
3. 尝试其他浏览器（Chrome、Firefox、Edge）

**Q: 颜色看起来不对？**
A: 这取决于你的屏幕亮度和色彩设置。可以在代码中调整：

- 蓝色：改 `#6366f1` 为你喜欢的颜色
- 紫色：改 `#8b5cf6`
- 等等

**Q: 怎样改颜色？**
A: 编辑 `src/views/Home.vue` 和 `src/views/Layout.vue`，找到 `#6366f1` 或 `#8b5cf6`，替换为你的颜色。
